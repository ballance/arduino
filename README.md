# Bash's Arduino Projects 🤖

Welcome to your very own Arduino lab! Each folder is one project.
Do them in order — every project teaches you something new.

## The Projects

| # | Project | What you'll learn |
|---|---------|-------------------|
| 1 | [Traffic Light](01-traffic-light/) | Turning LEDs on and off, and using `delay()` to wait |
| 2 | [Random Flash](02-random-flash/) | Making the Arduino pick random numbers, like rolling dice |
| 3 | [Button Light](03-button-light/) | Reading a button — your first input! — and making choices with `if` |
| ⚡ | [Circuit School](circuit-school/) | How circuits really work: loops, breadboards, LEDs, resistors. **Read before Project 4!** |
| 4 | [Circuit Drills](04-circuit-drills/) | Design circuits from scratch — no wiring tables, you're the engineer |

Starting with Project 4, you build the circuits **yourself**. Projects
give you the goal and the parts; the wiring is up to you. (Dad has
the answer keys.)

## How to put a program on your Arduino

1. Plug the Arduino Uno into the computer with the USB cable.
2. Open the **Arduino IDE** app.
3. Go to **File → Open** and pick the project's `.ino` file
   (for example `01-traffic-light/01-traffic-light.ino`).
4. At the top of the window, click the board dropdown and pick
   **Arduino Uno** (and the port that shows up when you plug it in).
5. Click the **→ Upload** button (the arrow). The lights on the board
   will blink while it loads.
6. Watch your program run!

## If something doesn't work

- **LED not lighting up?** LEDs only work one way. Try flipping it around.
- **Nothing uploads?** Check the USB cable is plugged in all the way,
  and that you picked the right port.
- **Still stuck?** Ask Dad. 😄

## Rules of the lab

- Always unplug the Arduino before changing wires.
- It's okay to break things (in code). That's how you learn!
- When you finish a project, try the **challenges** at the bottom of
  its README before moving on.
